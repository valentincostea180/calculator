# 🧮 Calculator (JavaScript & HTML)

A browser-based calculator built with JavaScript, featuring interactive UI elements and basic arithmetic operations.


## 📌 Overview

The application allows users to perform calculations through button inputs, supporting chained operations and real-time display updates.


## ⚙️ Features
•	Basic operations: addition, subtraction, multiplication, division
•	Interactive button UI with hover effects


## ▶️ How to Run
1.	Open the project in a browser
2.	Click buttons to input numbers and operations

## 🧠 Concepts Used
•	DOM manipulation
•	Event listeners
•	Basic state management
•	Number parsing and calculations
